# 🚀 Void Runner

Infinite procedural tunnel runner with WebGL shaders & motion blur.

## Features
- Procedurally generated infinite tunnel
- WebGL shader effects (bloom, chromatic aberration)
- Motion blur at high speed
- Score tracking

## Tech
- WebGL / GLSL shaders
- Vanilla JavaScript

## Run
Open `index.html` in any modern browser. Requires WebGL support.
